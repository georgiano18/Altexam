Project Title: Deploying a Basic Web Server on AWS EC2

Student ID: ALT/SOE/024/1224

Project Description: This project involves setting up and deploying a basic web server on an Amazon Web Services (AWS) EC2 instance. It explores the fundamentals of cloud computing, server management, and web hosting. The goal is to understand how to configure, secure, and manage a server in the cloud, providing hands-on experience with AWS services including EC2, VPC, and Security Groups. This project not only enhances my technical skills but also provides a practical foundation for web development and cloud architecture.
